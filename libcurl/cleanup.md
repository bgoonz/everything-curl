<span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="9d4e3a466da344b89c187c38a3052b95"><span data-offset-key="9d4e3a466da344b89c187c38a3052b95:0">In previous sections we have discussed how to setup handles and how to drive the transfers. All transfers will, of course, end up at some point, either successfully or with a failure.</span></span></span>

<span class="text-4505230f--HeadingH700-04e1a2a3--textContentFamily-49a318e1"><span data-key="029f4ec1da4e425cb38b075144cf9392"><span data-offset-key="029f4ec1da4e425cb38b075144cf9392:0">Multi API</span></span></span>

<span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="e7b0c70e3573474e8cfbfe105f8b7b31"><span data-offset-key="e7b0c70e3573474e8cfbfe105f8b7b31:0">When you have finished a single transfer with the multi API, you use </span><span data-offset-key="e7b0c70e3573474e8cfbfe105f8b7b31:1">`curl_multi_info_read()`</span><span data-offset-key="e7b0c70e3573474e8cfbfe105f8b7b31:2"> to identify exactly which easy handle was completed and you remove that easy handle from the multi handle with </span><span data-offset-key="e7b0c70e3573474e8cfbfe105f8b7b31:3">`curl_multi_remove_handle()`</span><span data-offset-key="e7b0c70e3573474e8cfbfe105f8b7b31:4">.</span></span></span>

<span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="57b7a9993dab49efac5f156446b67d66"><span data-offset-key="57b7a9993dab49efac5f156446b67d66:0">If you remove the last easy handle from the multi handle so there are no more transfers going on, you can close the multi handle like this:</span></span></span>

    curl_multi_cleanup( multi_handle );

<span class="text-4505230f--HeadingH700-04e1a2a3--textContentFamily-49a318e1"><span data-key="cbd69183bb424c1ea07d0e6830ce76e8"><span data-offset-key="cbd69183bb424c1ea07d0e6830ce76e8:0">easy handle</span></span></span>

<span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="437f477d5d31482db8ac2c60718d8653"><span data-offset-key="437f477d5d31482db8ac2c60718d8653:0">When the easy handle is done serving its purpose, you can close it. If you intend to do another transfer, you are however advised to rather reuse the handle rather than to close it and create a new one.</span></span></span>
